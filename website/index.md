# HorseMD — 一个窗口，装下所有 .md 文件

> 免费 · 开源 · 不要账号

一个免费的 Typora 平替，但不止于此。

- **下载 Windows 版**（.exe，NSIS）/ **下载 macOS 版**（.dmg，Apple Silicon & Intel）/ **下载 Linux 版**（.deb）/ **下载 Android 版**（.apk）：https://github.com/BND-1/horseMD/releases/latest
- 构建未签名 — Windows：更多信息 → 仍要运行 · macOS：右键 → 打开 · Linux：sudo dpkg -i 安装 · Android：允许安装未知应用

## 它能做什么

| # | 功能 | 说明 | 快捷键 |
| --- | --- | --- | --- |
| 01 | 标签页 | 双击一个文件，是多一个标签，不是多一个窗口。 | Ctrl/⌘ Tab |
| 02 | 文件夹工作区 | 整个文件夹挂在侧边栏，新建、重命名、删除都不用切出去。 | Ctrl/⌘ B |
| 03 | 所见即所得 | 打字就渲染。表格、代码高亮、LaTeX、Mermaid、任务清单都认。 | / |
| 04 | 富文本复制 | 复制自带格式，粘进微信公众号、邮件、Notion 都不丢样式。 | Ctrl/⌘ C |

更多：命令面板（Ctrl+P）、大纲面板（点标题即跳转）、源码模式（切换保持位置）、分屏、查找替换、PDF 导出、Mermaid 全屏灯箱、设置页（排版/主题/拼写/图床）、自定义主题（可迁移 Typora 主题）、标签拖拽排序、会话恢复、外部修改自动重载、中英双语界面。

## 六套主题

明亮、暗夜，以及四套莫兰迪：雾、鼠尾草、玫瑰、暮色。
