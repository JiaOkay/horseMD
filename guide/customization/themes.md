# 主题

<span class="version-badge">适用于 HorseMD v0.6.0</span>

点击状态栏的主题名称，或进入“设置 → 外观”。HorseMD 提供暖光、暖夜和四套莫兰迪主题。主题会改变应用外壳、编辑区、菜单、查找高亮、代码块和弹窗的颜色。

按 `Ctrl+Shift+T` 可以循环切换内置主题。

## 导入自定义主题

在主题菜单或设置页点击“打开主题文件夹”，将 `.css` 文件或完整主题文件夹复制进去，再刷新主题列表。HorseMD 会递归查找 CSS，因此可以保留 Typora 主题原本的目录和字体资源。

选择自定义主题后：

- 正文区域采用主题的字体、颜色、宽度和背景。
- HorseMD 外壳仍使用当前应用主题，保持标签、菜单和按钮可读。
- CSS 中的相对 `url(...)` 会按主题文件位置解析，可加载同目录字体和图片。

::: warning 第三方 CSS
自定义主题可能覆盖正文布局或使用不存在的字体。出现文字不可见、宽度异常或资源丢失时，先切回内置主题，再检查 CSS。
:::
