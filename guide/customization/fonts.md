# 文档与代码字体

<span class="version-badge">适用于 HorseMD v0.6.0</span>

打开“设置 → 排版”，顶部有两个独立选择器：

- 文档字体：影响正文和标题。
- 代码字体：影响围栏代码块、CodeMirror 和行内代码的等宽显示。

![HorseMD 设置页中的字体和排版选项](/images/v0.6.0/settings.png)
<span class="image-caption">选择器、字号、行距、段距和页宽会在右侧示例中实时预览。</span>

## 选择字体

点击字段后可以搜索本机字体。列表中的“永和九年 Aa 123”和 `const x = 'hi'` 会使用候选字体自身渲染。鼠标悬停时，设置预览和当前编辑器临时应用该字体；点击后正式保存。选择“默认”恢复系统字体栈。

浏览器会请求本地字体访问权限，该权限只用于枚举字体名称和预览，不会读取文档或把字体上传到网络。

## 字体没有出现

确认字体已经安装到系统并允许当前用户使用，然后重新打开设置。某些字体只有单一字重，标题粗体可能由系统合成。代码字体建议选择明确支持 ASCII 标点的等宽字体。

设置页底部提供方正字库和 Nerd Fonts 外部链接；字体许可由对应提供者决定。
