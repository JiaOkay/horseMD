# Android

<span class="version-badge">适用于 HorseMD v0.6.0</span>

先按照 [Android 安装](/getting-started/install-android)完成 APK 安装。

## 打开和创建文件

点击顶栏加号新建文档。打开文件会调用 Android 系统文件选择器，选中的 Markdown 会导入 HorseMD 的应用文档库。其他应用分享 `.md` 到 HorseMD 时，也会导入并打开。

## 保存位置

文档保存在应用管理目录中，可以从 HorseMD 文件树访问。Android 对 `Android/data` 有访问限制，系统文件管理器看不到并不代表文件丢失。

需要把文档拿到外部时，使用底部“导出”打开系统分享面板，保存到公共目录、网盘、邮件或聊天应用。

## 触摸操作

侧边栏以抽屉形式覆盖正文，点击遮罩关闭。双击或长按文字只显示 Android 系统文本菜单，不显示桌面选区格式栏。手机屏幕使用全宽编辑，平板和横屏会保持舒适阅读宽度。

::: danger 卸载前导出重要文件
Android 卸载应用时可能删除应用文档库。重要文档应提前分享或备份到外部位置。
:::
