# iPhone 与 iPad

<span class="version-badge">适用于 HorseMD v0.6.0 开发安装版</span>

iOS 版与桌面版共享富文本编辑器，但当前没有公开 App Store 或 TestFlight 安装包，需要使用 Xcode 和 Apple ID 安装到自己的设备。免费 Apple ID 安装的开发应用通常需要定期重新签名。

## 界面操作

- 顶栏菜单按钮打开侧边抽屉。
- 抽屉中切换主页、文件树和大纲。
- 顶栏加号新建文档，命令按钮打开命令面板。
- 底栏显示保存状态、字数、导出和更多操作。

点击文件或大纲标题后，抽屉会自动收起。双击或长按文字时只显示 iOS 系统选择菜单，复制、粘贴、全选、查词和无障碍操作保持原生体验，不再叠加桌面格式工具栏。

## 保存与分享

保存后的文件可以在系统“文件 → 我的 iPhone/iPad → HorseMD”中找到。需要复制到 iCloud Drive、AirDrop 或其他应用时，点击底部“导出”。

iOS 不显示分屏、桌面 PDF 导出、访达定位、自定义图床命令和窗口控制等桌面功能。
