# 工作区与文件夹

<span class="version-badge">适用于 HorseMD v0.6.0</span>

工作区是左侧文件树的容器，可以同时挂载多个磁盘文件夹。添加新文件夹不会替换已有文件夹，也不会把它们复制到 HorseMD 内部。

![包含多个目录的 HorseMD 工作区](/images/v0.6.0/workspace.png)
<span class="image-caption">根目录是工作区挂载点；其下文件和子目录可以展开、创建、重命名和删除。</span>

## 添加文件夹

1. 点击活动栏的文件夹图标。
2. 空工作区点击“添加文件夹”；已有内容时点击侧边栏顶部的文件夹加号。
3. 在系统对话框中选择目录。
4. 重复操作可以继续添加其他目录。

根目录本身不会被 HorseMD 重命名、拖走或删除。点击根目录右侧菜单可以从工作区移除它；“移除”只取消挂载，不会删除磁盘文件。

## 文件操作

在文件或文件夹上右键，可以新建、重命名、复制、打开所在位置或删除。删除使用系统废纸篓/回收站。拖动普通文件和子目录可以在同一工作区内移动，但不能把根挂载点当作普通文件夹移动。

侧栏顶部的四个按钮分别用于：添加外部文件夹、新建文件、新建子文件夹，以及展开或折叠全部目录。添加文件夹使用带加号的文件夹图标，新建文件夹使用独立的目录创建图标。

## 自动恢复与监听

工作区路径会随会话保存，重新打开 HorseMD 后自动恢复。已挂载目录发生新增、删除或重命名时，文件树会自动刷新。系统根目录和受限路径不能被添加，以避免权限错误和大规模文件监听。
