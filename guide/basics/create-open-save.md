# 新建、打开与保存

<span class="version-badge">适用于 HorseMD v0.6.0</span>

## 新建文档

点击顶栏加号，或按 `Ctrl/Cmd+N`。新文档会提供一级标题和正文位置，可以立即输入。没有路径的标签属于临时草稿，标题通常显示为“未命名”。

## 打开已有文件

使用以下任一方式：

- 按 `Ctrl/Cmd+O` 选择一个或多个文件。
- 从工作区文件树点击 Markdown 文件。
- 从访达或资源管理器双击已关联的 `.md` 文件。
- 将启动文件交给 HorseMD 打开。

`.md`、`.markdown` 和 `.mdx` 默认使用富文本编辑器。`.txt` 等普通文本文件使用快速源码编辑器。特别大的 Markdown 会先以快速源码模式打开，避免整篇解析造成长时间卡顿；需要时可以主动进入富文本。

## 保存状态

底部状态栏会显示：

- “已保存”：当前内容与磁盘一致。
- “已修改”：存在尚未写入磁盘的修改。

模式切换、初始化恢复和程序内部同步不会触发“已修改”，只有真实编辑才会改变保存状态。

按 `Ctrl/Cmd+S` 保存，按 `Ctrl/Cmd+Shift+S` 另存为。关闭未保存标签或退出应用时，HorseMD 会要求确认。

::: tip 外部修改
如果其他程序修改了当前文件，HorseMD 会通过文件监听刷新内容；当当前标签还有未保存修改时，不会直接覆盖你的编辑。
:::
