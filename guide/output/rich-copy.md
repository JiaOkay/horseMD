# 富文本复制

<span class="version-badge">适用于 HorseMD v0.6.0</span>

在富文本模式选中内容并按 `Ctrl/Cmd+C`。HorseMD 会同时提供纯文本和带内联样式的 HTML，目标应用可以选择自己支持的格式。

适合粘贴到：

- 微信公众号编辑器。
- 邮件正文。
- Notion、语雀和支持富文本粘贴的知识库。
- Word 或其他办公软件。

标题、段落、列表、引用、表格、代码和链接会尽量保留当前主题下的可读样式。编辑器专用控件不会被复制，例如代码块复制按钮、表格手柄和块级菜单。

## 只需要 Markdown 源码

切换到源码模式后再复制，得到的是原始 Markdown 文本。也可以直接从磁盘读取 `.md` 文件。

::: tip 目标平台会二次处理样式
富文本复制不能强制目标应用接受所有 CSS。发布前应在目标平台预览，尤其检查表格宽度、代码背景、图片和字体。
:::
