# 图片与图床

<span class="version-badge">适用于 HorseMD v0.6.0</span>

## 插入图片

可以将图片粘贴到正文、拖入编辑器，或通过斜杠菜单插入图片。代码块获得焦点时，粘贴和拖入保持代码输入行为，不会被图片处理拦截。

本地相对路径会以当前 Markdown 文件所在目录解析。例如：

```md
![界面截图](./images/interface.png)
```

移动 Markdown 文件时，应同时移动相对路径引用的图片目录。网络图片使用 `https://` 或受信任的 `http://` 地址，加载速度取决于远程服务器。

## 查看大图

双击正文图片可以打开灯箱，按 Escape 或点击背景关闭。图片说明使用 Markdown 图片的替代文字；替代文字也有助于图片加载失败和无障碍阅读。

## 自定义图床

点击顶栏图片按钮，或进入“设置 → 图床”，填写类似 `picgo upload` 的上传命令。配置后，HorseMD 会把粘贴、拖入或选择的图片写入临时文件，执行：

```text
<你的命令> "<临时图片路径>"
```

命令最后输出的 `http://` 或 `https://` 地址会作为图片链接插入。按钮上的强调色圆点表示已经配置。

::: warning 先在终端验证命令
HorseMD 不负责安装 PicGo 或其他上传工具。命令应在终端中可以独立运行，并且只把可信 URL 输出到标准输出。留空会恢复默认本地图片行为。
:::
