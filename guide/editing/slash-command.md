# 斜杠命令

<span class="version-badge">适用于 HorseMD v0.6.0</span>

在富文本正文中输入 `/`，会打开快捷菜单。继续输入内容即可筛选，按上下方向键选择，按 Enter 执行，按 Escape 关闭。

![使用拼音首字母匹配标题命令](/images/v0.6.0/slash-command.png)
<span class="image-caption">输入 `/bt` 后匹配“标题”，菜单始终保持在窗口可视区域内。</span>

## 可以怎样搜索

菜单支持中文名称、英文名称、常用别名、完整拼音和拼音首字母。例如：

| 输入 | 可以找到 |
| --- | --- |
| `/表格`、`/bg` | 表格 |
| `/标题`、`/bt` | 标题 1～6 |
| `/wxlb` | 无序列表 |
| `/quote` | 引用 |
| `/javascript`、`/python` | 已设置语言的代码块 |
| `/mermaid` | Mermaid 代码块 |

短关键词优先精确和前缀匹配，减少无关结果。没有结果时菜单会显示空状态，不会在输入过程中突然消失。

## 菜单位置

在窗口顶部或底部输入时，菜单会根据可用空间向下或向上展开，并限制最大高度。结果较多时在菜单内滚动，不会被应用边框裁掉。

斜杠菜单只在富文本普通段落中工作。在代码块中输入 `/` 会作为代码内容，不会抢占键盘输入。
