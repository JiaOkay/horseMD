# 快捷键

<span class="version-badge">适用于 HorseMD v0.6.0</span>

表格中的 `Ctrl/Cmd` 表示 Windows 使用 Ctrl、macOS 使用 Command。标题层级快捷键固定使用 Ctrl。

## 文件与标签

| 操作 | 快捷键 |
| --- | --- |
| 新建文档 | `Ctrl/Cmd+N` |
| 打开文件 | `Ctrl/Cmd+O` |
| 打开文件夹 | `Ctrl/Cmd+Shift+O` |
| 保存 | `Ctrl/Cmd+S` |
| 另存为 | `Ctrl/Cmd+Shift+S` |
| 关闭标签 | `Ctrl/Cmd+W` |
| 下一个标签 | `Ctrl+Tab` |
| 上一个标签 | `Ctrl+Shift+Tab` |

## 编辑与导航

| 操作 | 快捷键 |
| --- | --- |
| 命令面板 | `Ctrl/Cmd+P` |
| 查找 | `Ctrl/Cmd+F` |
| 切换侧边栏 | `Ctrl/Cmd+B` |
| 切换大纲 | `Ctrl+Shift+L` |
| 切换源码模式 | `Ctrl/Cmd+/` |
| 切换主题 | `Ctrl+Shift+T` |
| 标题 1～6 | `Ctrl+1`～`Ctrl+6` |
| 转为正文 | `Ctrl+0` |
| 黄色高亮 | `Ctrl/Cmd+Alt+H` |

## 输出

| 操作 | 快捷键 |
| --- | --- |
| 导出 PDF | `Ctrl/Cmd+Shift+E` |
| 复制 | `Ctrl/Cmd+C` |

`Ctrl/Cmd+B` 在 HorseMD 中用于侧边栏，不用于粗体。粗体可以通过选区工具栏或 `**文字**` 语法设置。
