# Android 安装

<span class="version-badge">适用于 HorseMD v0.6.0</span>

Android 版与桌面版共享编辑器，但文件选择、保存和分享遵循 Android 系统规则。

## 安装 APK

1. 下载 `HorseMD-0.6.0.apk`。
2. 在浏览器下载列表或系统文件管理器中点击 APK。
3. 首次侧载时，Android 会要求允许当前浏览器或文件管理器“安装未知应用”。
4. 临时开启该权限，返回并继续安装。
5. 安装完成后打开 HorseMD。

系统所说的“未知应用”表示 APK 不是从应用商店安装，不等于文件有问题。安装完成后，可以关闭刚才授予浏览器或文件管理器的安装权限。

## 文件保存位置

Android 版会把文档保存到应用管理的目录。由于系统存储限制，一些文件管理器无法直接浏览 `Android/data`。在 HorseMD 内可以通过文件树查看这些文档；需要发送到其他应用或保存到公共目录时，使用底部“导出/分享”。

卸载 Android 应用可能同时删除应用目录中的文件。重要文档应定期通过系统分享面板导出到网盘、公共文件夹或其他设备。

详细操作见 [Android 使用指南](/mobile/android) 和[移动端分享](/output/mobile-share)。
