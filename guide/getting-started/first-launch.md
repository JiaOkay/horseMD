# 第一次启动

<span class="version-badge">适用于 HorseMD v0.6.0</span>

第一次打开 HorseMD 时，会自动出现“欢迎使用 HorseMD”文档。这是一个可直接阅读和操作的本地引导页，只在全新会话中自动创建一次。

![HorseMD 第一次启动时显示欢迎文档](/images/v0.6.0/first-launch.png)
<span class="image-caption">欢迎文档介绍大纲、文件树、快捷键和基础操作。点击图片可以放大。</span>

## 完成第一次操作

1. 点击顶栏右侧的加号，新建文档。
2. 输入标题和一段正文。
3. 按 `Ctrl/Cmd+S`，选择保存位置和文件名。
4. 点击左侧文件夹图标，选择“添加文件夹”。
5. 添加你平时存放 Markdown 的文件夹。

保存后的文档会显示真实文件名；底部状态从“已修改”恢复为“已保存”。未命名的新文档如果有内容，也会在正常退出后作为草稿恢复，但草稿不等于磁盘备份，重要内容仍应及时保存。

## 欢迎文档去了哪里

欢迎文档是普通的临时标签页，可以关闭。以后需要查阅操作方式时，直接访问本教程；关闭欢迎页不会删除任何本地文件。

下一步建议阅读[认识界面](/basics/interface)和[新建、打开与保存](/basics/create-open-save)。
