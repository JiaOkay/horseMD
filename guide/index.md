## 推荐阅读顺序

1. [下载正确的版本](/getting-started/download)，完成对应平台的安装。
2. 阅读[认识界面](/basics/interface)，了解工作区、标签页、编辑区和状态栏。
3. 通过[新建、打开与保存](/basics/create-open-save)完成第一篇文档。
4. 根据工作需要查阅编辑、效率工具、输出和自定义章节。

::: tip 不必一次读完
除了“开始使用”部分，其余页面都可以独立阅读。页面右上角的搜索能够直接查找功能名称、按钮文字和快捷键。
:::

## 教程维护原则

教程中的界面截图来自对应版本的正式安装应用。HorseMD 发布用户可见功能时，会同步检查相关教程；页面顶部的适用版本和底部的更新时间可以帮助你判断内容是否仍然适用。

发现教程与实际界面不一致时，可以使用页面底部的“在 GitHub 上改进本页”，或者前往 [Issues](https://github.com/BND-1/horseMD/issues) 反馈。
