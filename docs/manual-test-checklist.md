# 功能清单 + 手动测试清单(重构回归基线)

> 重构的安全网。每条 = 一个可手动复现的步骤 → 预期。重构每抽一块,跑相关
> 条目;全部重构完后做整表回归。勾选框 `[ ]` 供手动执行时打勾。
> 平台:macOS arm64(`/Applications/HorseMD.app`)为主,Windows/移动在分节标注。

---

## A. 标签页 / 窗口
- [ ] 双击文件树多个 md → 各开一个标签,只有一个窗口
- [ ] 切换标签 → 编辑器跟随,已挂载的编辑器不重建(秒切)
- [ ] 关闭有未保存内容的标签 → 弹确认;关闭无内容的 → 直接关
- [ ] 关闭最后一个文件 → 进 Home,大纲清空(#20)
- [ ] 拖拽标签?/ 右键「在分屏打开」→ 右格出现,两格都可编辑
- [ ] macOS 关窗口(红绿灯)→ 进程保留;Cmd+Q 有未保存 → 弹确认
- [ ] Windows:最小/最大/关闭按钮工作;最大化图标状态同步

## B. 文件树 / workspace
- [ ] 打开文件夹 → 侧边栏树展开;增删改文件实时刷新(watcher)
- [ ] 外部改了已打开文件 → 编辑器提示重载(文件监听回显)
- [ ] 右键文件:重命名 / 删除 / 在 Finder 显示 / 导出 PDF
- [ ] 深层目录可展开/折叠;文件多的大文件夹不卡死(性能项)

## C. 命令面板 / 快捷键
- [ ] ⌘P → 命令面板,可搜文件 + 命令
- [ ] ⌘B → 切侧栏;⌘F → 查找;⌘⌥F → 替换(聚焦替换框)
- [ ] ⌘S → 保存;⌘/ → 源码/富文本切换;⌘P 命令齐全(find/replace/theme/source/outline/files)

## D. 源码 ↔ 富文本
- [ ] .md 默认富文本;.txt 默认纯文本(textarea)
- [ ] ⌘/ 切源码 → 显示 markdown 源码;再切回 → 富文本,内容不丢
- [ ] 源码模式编辑 → 保存 → 内容正确(非受控 textarea 同步)

## E. 大纲
- [ ] 侧栏切「大纲」→ 列出 H1–H6
- [ ] 滚动文档 → 大纲高亮跟随当前标题(reflow-free scrollspy),不卡、停得准
- [ ] 大纲面板软居中:active 项漂到上下边沿 → 自动滚到中间;滚到文档底 → 最后标题也到中间
- [ ] 点大纲项 → 文档跳到该标题
- [ ] 关闭文件 → 大纲清空(#20)

## F. 主题 / 自定义主题
- [ ] 切 light/dark + 6 套内置主题(含莫兰迪)
- [ ] 放 Typora 风格 `.css` 到 userData/themes → 被识别、可应用、相对 url() 生效
- [ ] 切自定义主题后 app chrome 仍是自己的样式,写作区用主题

## G. 国际化
- [ ] 切中文/英文 → 所有 UI 文本、命令面板、斜杠菜单、查找替换、保存提示都跟随

## H. 查找替换(#19)
- [ ] ⌘F → 查找栏(带替换行,默认展开)
- [ ] 输入查找词 → 实时高亮 + `n/m` 计数;▲▼ 上一个/下一个(循环)
- [ ] 输入替换词 →「替换」替当前并跳下一个;「全部」替换全部
- [ ] 替换框 Enter=替一个,Shift+Enter=全部;Esc=关
- [ ] **富文本模式**:替换正确(块内、多匹配)
- [ ] **源码模式(⌘/)**:替换正确(非受控 textarea 适配)← 重点回归
- [ ] 无查找词时替换按钮 disabled

## I. 保存 / 保存提示
- [ ] ⌘S / 浮动保存按钮 → 写盘,桌面端弹「已保存 ✓」(~1.5s 自隐)
- [ ] 保存失败 → 弹红色 sticky 错误
- [ ] 新文件首次保存 → 另存为对话框
- [ ] 有图片的文档首次保存 → 图片搬到 ./assets/,链接改相对路径
- [ ] 未保存草稿 → 关闭/重启后恢复(session untitled)

## J. 导出 PDF
- [ ] 命令/右键导出 PDF → 生成 PDF,富文本样式(标题/代码/图片)正确
- [ ] 未打开的文件右键导出 → 自动打开挂载后再导

## K. 富文本基础(打字即渲染)
- [ ] 标题 H1–H6、有序/无序/任务列表、引用、链接、分隔线、粗/斜/删除线
- [ ] 表格:增删行列、单元格 Enter 换行(往返 `<br>`,#7)、源码往返不坏
- [ ] 行内代码:打到反引号后右边自动退出 code(不无限继承,#10)
- [ ] 斜杠 `/` → 块菜单(中文);块拖拽 handle 可用

## L. 代码块(CodeMirror)
- [ ] ``` 代码块 → 语法高亮 + 深色代码面 + Copy 按钮
- [ ] 代码块可编辑;Mermaid/LaTeX 块走预览机制(见 L1/M)

## L1. Mermaid
- [ ] ```mermaid → 默认渲染图,工具栏 Hide/Edit 切换源码
- [ ] 粘贴两段 mermaid 进同块 → 自动拆成两块;多个图各自渲染
- [ ] 主题切换 → mermaid 重渲染配色

## M. LaTeX 数学(#18 + 渲染修复 + 打字)
- [ ] `$$x^2+y^2$$` 单行 → 渲染成 **KaTeX 块公式**(不是裸代码块)
- [ ] `$$\n…\n$$` 多行 → 同样块公式
- [ ] 内联 `$E=mc^2$` → 内联 KaTeX
- [ ] **打字** `$$x^2$$`(单行,直接在段落里打)→ 变成块公式(不是行内)
- [ ] **打字** `$E=mc^2$` → 行内公式(单 `$` 不受块规则影响)
- [ ] 富文本↔源码往返 → 公式不坏
- [ ] Mermaid 和 LaTeX 同时存在 → 两者都渲染(renderPreview 链式未回退)

## N. 高亮 / 行内 HTML(#14)
- [ ] `==text==` → 黄色高亮;工具栏高亮按钮可选红/黄/蓝;⌥⌘H 切黄
- [ ] 行内 HTML `<span style=color:red>x</span>` → 渲染;块 `<table>` HTML → 渲染表格
- [ ] 源码往返保留原始 HTML

## O. Front matter
